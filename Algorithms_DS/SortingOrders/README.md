| Feature   | Bubble Sort | Quick Sort       |
| --------- | ----------- | ---------------- |
| Technique | Comparison  | Divide & Conquer |
| Best Case | O(n)        | O(n log n)       |
| Average   | O(n²)       | O(n log n)       |
| Worst     | O(n²)       | O(n²)            |
| Speed     | Slow        | Fast             |


Quick Sort is preferred because:

✅ Faster for large datasets
✅ Requires less extra memory
✅ Performs fewer comparisons
✅ Used in many real-world systems

Example:

E-commerce platforms
Databases
Search engines
